COMO USAR

1. Abra index.html
2. Entre em admin-login.html
3. Login padrão:
   usuário: admin
   senha: 1234

FUNÇÕES
- Salva tudo no localStorage do navegador
- Cria, edita, publica e exclui páginas
- Exporta backup em JSON
- Importa backup em JSON
- Tem visualizador para páginas publicadas

OBSERVAÇÃO
Os dados ficam apenas neste navegador/dispositivo até você exportar um backup.
